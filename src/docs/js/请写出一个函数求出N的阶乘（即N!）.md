# 请写出一个函数求出N的阶乘（即N!）

作者：![cleverboy32](https://avatars.githubusercontent.com/u/17595456?s=80&u=aa9d4327a05b27684339ac251eb81825edfe1b0a&v=4)[cleverboy32](https://github/cleverboy32)


```
    function factorial(n) {
          if (n > 1)  return n*factorial(n-1);
          return 1;
    }
    
    
```
